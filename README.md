# 🌍 World Explorer

A web application to explore countries and view basic information using a public API.

---

## 🎯 Purpose

The purpose of this project is to practice API integration, JavaScript, and basic UI development by building a simple and interactive web app.

---

## 🌐 API Used

This project uses the **REST Countries API** to fetch country data:

https://restcountries.com/v3.1/all?fields=name,flags,capital,region,population

---

## 🚀 Features

* View list of countries
* Search countries by name
* Display basic details (flag, capital, region, population)
* Responsive grid layout

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript (Fetch API)

---

## ▶️ How to Run

1. Clone the repository
2. Open the project folder
3. Open `index.html` in your browser
